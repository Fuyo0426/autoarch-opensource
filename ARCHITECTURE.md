## 核心引擎

### ArchGateway — 總發言人
接收所有 LINE / API 訊息的唯一入口。負責身份驗證、訊息格式化、回覆統一輸出。所有 Squad 的回答都經過這裡才送給用戶。

### PermissionManager — 權限管理
管理用戶角色（Guest / Basic / Mid / Pro / Admin）與月度配額。決定哪個用戶可以使用哪些 Squad 功能。

### SmartRouter — 意圖分類器
分析用戶訊息，判斷意圖屬於法規查詢、設計諮詢、財務試算還是其他類型，自動路由到對應 Squad。

### BossAgent — 任務調度
將複雜任務拆解後分配給多個 Squad 並行處理，最後整合各 Squad 回覆，輸出完整答案。

### EvolutionCore — 自我進化
動態掃描 `skills/` 與 `squads/` 目錄，熱載入新功能。任何人新增一個 `.py` 檔案，系統下次啟動即自動啟用。

### S03 法規智慧組

> 建築法規的大腦。解讀條文、計算容積、解析地籍，是所有設計決策的法規底線。

| 成員 | 角色 | 職責說明 |
|------|------|---------|
| M01 | 法務長 | 統合法規審查，產出開發可行性報告 |
| M02 | 法規分析師 | 解讀建築法、都計法，計算容積率、建蔽率 |
| M03 | 地政專員 | 解析地籍謄本、判讀地籍圖、確認使用分區 |
| M04 | 法規搜尋員 | 對 ChromaDB 執行 RAG 向量語義搜尋 |
| M05 | 法規審核員 | 紅隊糾錯，對 AI 回答進行品質評分與修正 |
| M06 | 數位化專員 | 將新法規 PDF 自動 OCR 入庫至向量資料庫 |

**可延伸方向**
- 新增縣市別法規解讀員（台北市/新北市/台南市...）
- 新增歷史法規版本對照員

## 延伸指南

```
| M07 | 新角色名稱 | 一句話說明職責 |
```

### 新增 Squad

### 新增系統功能
