步驟一

下載 GitHub Desktop

這是一個視覺化工具，讓你完全不需要打指令，就能操作 Git。

1

安裝應用程式

前往官網下載，Mac / Windows 都支援。
安裝後用 GitHub 帳號登入。

desktop.github.com →

步驟二

把專案 Clone 到電腦

2

Clone 是什麼？

把 GitHub 上的資料夾，複製一份到你的電腦。
之後你的修改只在本機，確認好了再同步。

GitHub Desktop → File → Clone Repository
輸入：Fuyo0426 / autoarch-opensource
選擇存放位置 → Clone

步驟三

修改檔案，Push 上去

3

就像存 Word 一樣

在自己的 Squad 資料夾新增或修改 .md 檔案後：

①

GitHub Desktop 自動偵測到你的變更（左側列表）

②

左下角輸入說明 → 按 Commit to main

③

右上角按 Push origin → 完成

寫說明的格式

一行說清楚，系統自動記錄。

  • [S03] 新增都市計畫法第 85 條解讀
  • [S04] 更新停車位計算公式
  • [S06] 修正 BIM LOD 審查標準
  • [S11] 新增 2026 建材造價基準

格式：[S + Squad號] 然後用中文說你做了什麼。不需要很精確，說清楚就好。

你 Push 之後

系統自動做這些事。

索引更新

自動整理

所有法規文件、技能清單，自動重新編目。不需要任何人手動維護。

進化紀錄

自動追蹤

系統規模、貢獻者、每次更新，全部記錄在 EVOLUTION.md。

CHANGELOG

自動日誌

每次 push 的說明自動整理成版本歷史，往後可以查每個版本的進化過程。
