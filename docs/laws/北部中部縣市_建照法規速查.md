# 北部及中部縣市建照法規速查

> **文件用途**：供 AI RAG 系統索引使用，整合基隆市、桃園市、新竹市、新竹縣、苗栗縣、彰化縣、南投縣七縣市都市計畫建蔽率、容積率及建照申請規定。
> **資料基準日**：2026-03-26
> **法源基礎**：都市計畫法、都市計畫法臺灣省施行細則（內政部）、各縣市都市計畫書圖及土地使用分區管制要點
> **注意**：各都市計畫區管制要點具有個別效力，實際申請以各縣市政府最新公告版本為準。

## 通用基礎規定（全國適用）

### 都市計畫法臺灣省施行細則建蔽率上限（第32條）

| 使用分區 | 建蔽率上限 |
|---------|-----------|
| 住宅區 | 60% |
| 商業區 | 80% |
| 工業區 | 70% |
| 行政區 | 60% |
| 文教區 | 60% |
| 體育運動區 | 60% |
| 倉庫區 | 70% |
| 風景區 | 20% |
| 保護區 | 10% |
| 農業區 | 10% |
| 保存區 | 60% |

> 法源：[都市計畫法臺灣省施行細則](https://glrs.moi.gov.tw/LawContent.aspx?id=GL000452)（內政部）

### 都市計畫法臺灣省施行細則容積率基準（第34條）

各縣市住宅區及商業區容積率依計畫容納密度分級訂定，以下為參考級距：

| 居住密度 | 住宅區容積率範圍 | 商業區容積率範圍 |
|---------|----------------|----------------|
| 未達 200 人/公頃 | 120% ~ 150% | 180% ~ 210% |
| 200 ~ 300 人/公頃 | 150% ~ 180% | 210% ~ 240% |
| 300 ~ 400 人/公頃 | 180% ~ 200% | 240% ~ 280% |
| 400 人/公頃以上 | 200% ~ 240% | 280% ~ 320% |

其他分區固定容積率：

| 分區 | 容積率 |
|------|--------|
| 工業區 | 210% |
| 文教區 | 250% |
| 行政區 | 250% |
| 倉庫區 | 300% |
| 風景區 | 60% |
| 保存區 | 160% |

> 說明：上述為省細則預設值，各縣市都市計畫書另訂有較嚴格規定者從其規定。

### 非都市土地各種建築用地規定（全國統一）

| 用地類別 | 建蔽率 | 容積率 | 適用說明 |
|---------|--------|--------|---------|
| 甲種建築用地 | 60% | 240% | 一般鄉村聚落住宅使用 |
| 乙種建築用地 | 60% | 240% | 農林漁牧從業人員住宅 |
| 丙種建築用地 | 40% | 120% | 山坡地保育區、風景區 |
| 丁種建築用地 | 70% | 300% | 工廠及工業相關設施 |
| 窯業用地 | 60% | 120% | 窯業生產設施 |
| 交通用地 | 40% | 120% | 交通設施 |
| 遊憩用地 | 40% | 120% | 遊憩設施 |
| 殯葬用地 | 40% | 120% | 殯葬設施 |

> 法源：[非都市土地使用管制規則](https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=D0060013)（全國法規資料庫）

## 附錄一：建照申請通用程序

| 7. 開工 | 取得建照後辦理開工申報 |
