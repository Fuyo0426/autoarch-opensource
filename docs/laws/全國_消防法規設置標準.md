# 全國消防法規設置標準

> 主管機關：內政部消防署
> 資料查核日期：2026-03-26（三次獨立查核：全國法規資料庫 × 內政部主管法規系統 × 國介企業法規資料）

## 一、各類場所分類定義（第12條）

| 類別 | 場所類型 | 主要用途特徵 |
|------|----------|------------|
| **甲類** | 電影院、歌廳、舞廳、夜總會、俱樂部、保齡球館、遊藝場、遊樂場、麻將館、撞球場、酒家、酒吧、PUB | 公眾娛樂聚集，火災危險性高 |
| **甲類（續）** | 旅館、飯店、招待所、三溫暖 | 住宿類，需24hr保護 |
| **甲類（續）** | 醫院、療養院、長期照顧機構、老人福利機構、安養機構、育嬰室 | 行動不便者聚集 |
| **甲類（續）** | 百貨公司、超級市場、量販店（1,500m²以上）、餐廳 | 大型商業聚集 |
| **乙類** | 車站、飛機場候機室、港埠候船室 | 大量人流轉運 |
| **乙類（續）** | 期貨業、金融機構、郵局、電信局 | 金融服務 |
| **乙類（續）** | 學校教室、補習班、訓練班、幼兒園 | 教育場所 |
| **乙類（續）** | 圖書館、博物館、美術館、科學館 | 文教場所 |
| **乙類（續）** | 辦公室、集合住宅、寄宿舍 | 一般使用 |
| **乙類（續）** | 體育館、活動中心、室內溜冰場、室內游泳池 | 休閒運動 |
| **乙類（續）** | 倉庫、汽車駕訓班、電視播送場 | 特定用途 |
| **丙類** | 電信機械室、汽車修護廠、飛機修理廠、飛機庫 | 設備機械場所 |
| **丙類（續）** | 室內停車場、建築物附設室內停車空間 | 停車場所 |
| **丁類** | 高度危險工作場所（大量可燃性物質） | 工業危險作業 |
| **丁類（續）** | 中度危險工作場所 | 一般工業作業 |
| **丁類（續）** | 低度危險工作場所（少量可燃物） | 輕度工業 |
| **戊類** | 複合用途建築物（含甲類用途者） | 複合型 |
| **戊類（續）** | 複合用途建築物（乙至丁類） | 複合型 |
| **戊類（續）** | 地下建築物 | 地下型態 |
| **己類** | 大眾運輸工具（船舶、飛機、火車、捷運等） | 交通載具 |

## 二、滅火器設置條件（第14條）

| 場所類別 | 設置門檻 | 備註 |
|----------|----------|------|
| 甲類場所 | **全部應設置**（無面積下限） | 高風險場所無條件設置 |
| 地下建築物 | **全部應設置** | |
| 幼兒園 | **全部應設置** | |
| 乙、丙、丁類場所 | 總樓地板面積 **≥150m²** | 合計總面積計算 |
| 地下層或無開口樓層 | 樓地板面積 **≥50m²** | 適用各類場所 |
| 高層建築物（11層以上） | **全部應設置** | |

**滅火器設置距離**：步行距離不得超過20公尺（甲類場所）或30公尺（其他）。

## 三、室內消防栓設備設置條件（第15條）

| 場所類別 | 設置門檻 | 備註 |
|----------|----------|------|
| 甲類場所 | 任何一層樓地板面積 **≥500m²** | 或地下層≥200m² |
| 乙類場所 | 總樓地板面積 **≥1,000m²** | |
| 丙類場所 | 總樓地板面積 **≥1,000m²** | |
| 丁類場所 | 總樓地板面積 **≥1,000m²** | |
| 11層以上建築 | **全棟設置** | |
| 地下建築物 | **全部設置** | |

## 四、自動撒水設備設置條件（第17條）

| 建築物層數 | 場所類別 | 面積門檻 | 備註 |
|------------|----------|----------|------|
| 10層以下 | 甲類場所 | 樓地板面積合計 **≥300m²** | 核心觸發條件 |
| 11層以上 | 各類場所 | 任何一層 **≥100m²** | |
| 地下層 | 甲類場所 | 樓地板面積 **≥1,000m²** | |
| 無開口樓層 | 甲類場所 | 樓地板面積 **≥1,000m²** | |
| 高層建築物 | 全類 | **全棟設置**（無面積限制） | |
| 地下建築物 | 全類 | **全部設置** | |
| 總樓地板面積 | 甲類（特定） | **≥1,000m²** 含地下 | 含餐廳、量販店等 |

**自動撒水設備類型**：
| 類型 | 說明 | 適用場合 |
|------|------|----------|
| 閉式濕式 | 管內常壓蓄水，頭動水噴 | 一般場所 |
| 閉式乾式 | 管內充高壓空氣，頭動排氣後噴水 | 冷凍庫、低溫場所 |
| 開放式 | 管內無水，閥開後全區噴水 | 舞台等特殊場所 |
| 預動式 | 低壓空氣+流量偵測啟動 | 電信機房、電腦室 |

## 五、火警自動警報設備設置條件（第19條）

| 建築物層數 | 場所類別 | 設置條件 | 備註 |
|------------|----------|----------|------|
| 5層以下 | 甲類場所 | 任何一層樓地板面積 **≥300m²** | |
| 5層以下 | 乙、丙、丁類 | 總樓地板面積 **≥500m²** | |
| 6～10層 | 各類場所 | 任何一層樓地板面積 **≥300m²** | |
| 11層以上 | 全類 | **全棟設置**（無面積限制） | |
| 地下建築物 | 全類 | **全部設置** | |
| 地下層（獨立） | 各類場所 | 樓地板面積 **≥150m²** | |

**警報設備豁免條件**：已設置自動撒水設備（閉式噴頭，標示溫度≤75°C，60秒內動作），且涵蓋有效範圍內者，得免設火警自動警報設備。

**警報設備種類**：
- 火警自動警報設備（感熱/感煙探測器）
- 手動報警設備
- 緊急廣播設備
- 瓦斯洩漏火警自動警報設備

## 六、避難器具設置條件（第157條）

| 樓層位置 | 場所類別 | 收容人數 | 設置數量 |
|----------|----------|----------|----------|
| 2層以上（特定甲類） | 旅館、醫院、養護機構等 | 20～100人 | 1具 |
| 2層以上（特定甲類） | 旅館、醫院、養護機構等 | 每增加100人 | 增設1具 |
| 3層以上（一般甲類） | 餐廳、百貨公司等 | 50人以上 | 1具（每100人增設1具） |
| 地下層 | 甲類場所 | 任何人數 | 依規定設置 |

**避難器具種類**：緩降機、救助袋、滑台、避難繩索、避難橋、簡易緩降機。

**免設條件**：該樓層以具一小時以上防火時效牆壁區劃者；或建築物符合特定結構條件者。

## 七、室外消防栓設備（第16條）

| 條件 | 設置門檻 |
|------|----------|
| 適用場所 | 所有場所（依建築物規模） |
| 一般建築物 | 總樓地板面積 **≥1,500m²** |
| 甲、乙類場所 | 建築物高度 **>7m** 且總面積 **≥3,000m²** |

|----------|----------|

| 飛機修理廠、飛機庫 | **強制設置** |

## 九、消防審查申請程序

### 9.1 圖說審查（建造前）

```
步驟 1：起造人備妥文件
  ├── 消防安全設備設計圖說（依滅火→警報→避難順序繪製）
  ├── 建造執照申請書
  ├── 消防安全設備概要表
  └── 設計人資格證件

步驟 2：向當地消防機關提出申請
  ├── 台北市：台北市政府消防局
  ├── 新北市：新北市政府消防局
  └── 其他縣市：各縣市消防局

步驟 3：消防機關排定審查（掛號後通知）
  ├── 設計人攜帶資格證件出席
  └── 建築主管機關審訖圖說一併提供

步驟 4：審查結果處理
  ├── 合格 → 加蓋驗訖章，交起造人保存
  └── 不合格 → 一次告知不符事項 → 修正後複審

步驟 5：提交電子檔 PDF 備查
```

### 9.2 竣工查驗（使用執照前）

```
步驟 1：起造人提出竣工查驗申請
  ├── 消防安全設備測試報告書
  ├── 安裝施工測試佐證資料（含電子光碟）
  ├── 使用執照申請書
  └── 原審訖消防圖說

步驟 2：消防機關排定現場查驗日期

步驟 3：現場查驗
  ├── 合格 → 圖說標註「竣工圖」加蓋驗訖章
  └── 不合格 → 一次告知 → 改善後複查

步驟 4：完成列管（機關留存二份）
```

### 9.3 審查期限

| 建築物類型 | 審查期限 |
|------------|----------|
| 一般建築物 | 受理後 **7～10日** 內完成 |
| 供公眾使用建築物 | 得延長，最長 **不超過20日** |
| 構造複雜者 | 得延長，最長 **不超過20日** |

## 十、消防安全設備定期檢修申報

| 項目 | 規定 |
|------|------|
| 適用對象 | 一定規模以上之場所（甲類、公共場所） |
| 檢修頻率 | 每年一次（外觀、性能、綜合檢查） |
| 申報對象 | 向直轄市、縣（市）消防機關申報 |
| 執行人員 | 消防設備師或消防設備士 |
| 法源 | 消防安全設備檢修及申報辦法 |
