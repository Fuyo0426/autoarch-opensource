# 南部及東部離島縣市建照法規速查

> 涵蓋縣市：雲林縣、嘉義市、嘉義縣、屏東縣、宜蘭縣、花蓮縣、台東縣、澎湖縣、金門縣、連江縣（馬祖）
> 資料整理日期：2026-03-26
> 適用用途：AI RAG 系統建照法規查詢

---

## 前言：法規層級說明

台灣建照法規分三層：

| 層級 | 法規名稱 | 效力範圍 |
|---|---|---|
| 全國 | 建築法、非都市土地使用管制規則 | 全台適用 |
| 省/通用 | 都市計畫法臺灣省施行細則（D0070012） | 各縣市都市計畫區上限 |
| 地方 | 各縣市建築管理自治條例、細部計畫書 | 具體數值以此為準 |

**重要原則**：容積率、建蔽率的具體數值以各縣市核定之「都市計畫書圖」細部計畫管制要點為最終依據，省施行細則為上限規定。

## 全國通用基準：都市計畫法臺灣省施行細則

### 都市計畫區建蔽率上限（第32條）

| 使用分區 | 建蔽率上限 |
|---|---|
| 住宅區 | 60% |
| 商業區 | 80% |
| 工業區 | 70% |
| 行政區 | 60% |
| 文教區 | 60% |
| 體育運動區 | 60% |
| 風景區 | 20% |
| 保護區 | 10% |
| 農業區 | 10% |
| 保存區（古蹟） | 60%（原有建築已超過者從寬） |

### 都市計畫區容積率參考（第33、34條）

住宅區與商業區依計畫容納密度分級，未在計畫書載明時適用下列上限：

| 居住密度（人/公頃） | 住宅區容積率 | 商業區容積率 |
|---|---|---|
| 未達 200 | 120% 或 150% | 180% 或 210% |
| 200–300 | 150% 或 180% | 210% 或 240% |
| 300–400 | 180% 或 200% | 240% 或 280% |
| 400 以上 | 200% 或 240% | 280% 或 320% |

> 同一密度範圍有兩值者，依鄰里性公共設施用地佔比高低決定取高或取低值。

其他分區容積率上限：工業區 210%、行政區 250%、文教區 250%。

### 非都市土地使用管制規則（第9條）建蔽率與容積率

| 用地類別 | 建蔽率 | 容積率 | 備註 |
|---|---|---|---|
| 甲種建築用地 | 60% | 240% | 都市化地區農村住宅 |
| 乙種建築用地 | 60% | 240% | 鄉村區原住民等 |
| 丙種建築用地 | 40% | 120% | 山坡地、原住民保留地 |
| 丁種建築用地 | 70% | 300% | 工廠及工業設施 |
| 窯業用地 | 60% | 120% | |
| 交通用地 | 40% | 120% | |
| 遊憩用地 | 40% | 120% | |
| 殯葬用地 | 40% | 120% | |
| 特定目的事業用地 | 60% | 180% | |

> 各縣市政府得視實際需要「酌予調降」，並報內政部備查。

## 一、雲林縣

### 1. 都市計畫區使用分區建蔽率/容積率

雲林縣都市計畫區適用都市計畫法臺灣省施行細則，以下為各分區基準上限（具體數值依各鄉鎮細部計畫書為準）：

| 使用分區 | 建蔽率 | 容積率 |
|---|---|---|
| 住宅區 | ≤60% | 依細部計畫（一般 120%–240%） |
| 商業區 | ≤80% | 依細部計畫（一般 180%–320%） |
| 工業區（甲種） | ≤70% | ≤210% |
| 農業區 | ≤10% | 依計畫書 |
| 風景區 | ≤20% | ≤60% |
| 保護區 | ≤10% | 依計畫書 |

### 2. 非都市土地（雲林縣適用全國標準）

| 用地類別 | 建蔽率 | 容積率 |
|---|---|---|
| 甲種建築用地 | 60% | 240% |
| 乙種建築用地 | 60% | 240% |
| 丙種建築用地 | 40% | 120% |
| 丁種建築用地 | 70% | 300% |
| 特定目的事業用地 | 60% | 180% |

### 3. 特殊規定

- 雲林縣農業縣特性顯著，大量土地為「特定農業區」，建築用途嚴格受限
- 農業用地申請農舍須符合農業發展條例規定，建地面積不得超過申請面積10%（最高330平方公尺）
- 山坡地開發需提送水土保持計畫

### 4. 建照申請注意事項

- 主管機關：雲林縣政府建設處建築管理科
- 法規依據：雲林縣建築管理自治條例（民國105年1月25日）
- 申請建造執照須檢附：土地登記簿謄本、地籍圖謄本、位置圖、配置圖、各層平面圖、立面圖、剖面圖
- 特定建築物需附地質鑽探報告（二層以下樑跨6公尺以上鋼筋混凝土構造等）
- 審查期限：10至30日
- 建築線指定文件須載明：使用分區、建蔽率、容積率、退縮規定

## 二、嘉義市

- 建照申請文件：申請書、土地權利證明、建築線指示圖、工程圖樣（位置圖、地盤圖、各層平面圖、立面圖、剖面圖、屋頂平面圖）、結構計算書

|---|---|---|
| 甲種建築用地 | 60% | 240% |

| 丙種建築用地 | 40% | 120% |
| 丁種建築用地 | 70% | 300% |

### 3. 特殊規定

- **原住民族地區**：屏東縣原住民鄉鎮眾多（霧台、三地門、瑪家、泰武、來義、春日、獅子、牡丹等），原住民保留地上建築：
  - 申請人須具原住民身分
  - 建築基地面積不得超過330平方公尺
  - 山坡地保育區、風景區內變更為丙種建築用地（建蔽40%、容積120%）
  - 山坡地範圍外農業區變更為甲種建築用地（建蔽60%、容積240%）
- **恆春半島風景區**：墾丁國家公園範圍內建築受《國家公園法》管制，需另向墾丁國家公園管理處申請
- **農業區**：屏東縣為重要農業縣，特定農業區嚴格限制開發

### 4. 建照申請注意事項

- 主管機關：屏東縣政府城鄉發展處建築管理科
- 法規依據：屏東縣建築管理自治條例（民國103年8月26日）
- 建築線指定文件需載明建蔽率、容積率、退縮規定
- 墾丁國家公園範圍內需另取得國家公園許可
- 原住民鄉鎮土地需先確認土地類別（原住民保留地、一般農地、都市計畫地）

## 五、宜蘭縣

### 1. 都市計畫區使用分區建蔽率/容積率

宜蘭縣適用臺灣省施行細則，都市計畫區數值如下：

| 使用分區 | 建蔽率 | 容積率 |
|---|---|---|
| 住宅區 | ≤60% | ≤120%–240%（依密度） |
| 商業區 | ≤80% | ≤180%–320%（依密度） |
| 工業區 | ≤70% | ≤210% |
| 農業區 | ≤10% | 依計畫書 |
| 風景區 | ≤20% | ≤60% |

### 2. 非都市土地（宜蘭縣特別規定——低於中央標準）

> **重要注意**：宜蘭縣以自治條例將非都市土地建蔽率/容積率訂得低於中央標準，係歷任縣長環境保護政策延續（自游錫堃、呂國華縣長以來）。

| 用地類別 | 宜蘭縣自訂標準 | 中央標準（全國） |
|---|---|---|
| 甲種建築用地 | 建蔽60% / 容積**180%** | 建蔽60% / 容積240% |
| 乙種建築用地 | 建蔽60% / 容積**180%** | 建蔽60% / 容積240% |
| 丙種建築用地 | 建蔽40% / 容積**100%** | 建蔽40% / 容積120% |
| 丁種建築用地 | 建蔽70% / 容積**240%** | 建蔽70% / 容積300% |

> 宜蘭縣議會持續有爭議，部分議員主張應回復中央標準（容積率較低影響開發利益）。建議使用前確認最新版本自治條例。

### 3. 特殊規定

- **溫泉區**：礁溪、蘇澳等地設有溫泉專區，建築需符合宜蘭縣溫泉管理自治條例
- **農業環境保護**：宜蘭縣以「農業首都」定位，嚴格保護農地，農地變更開發程序嚴格
- **宜蘭縣整體保護方針**：「環境保護優先於開發」，非都市土地容積率全台最低之一
- **山坡地管制**：南澳鄉、大同鄉山坡地開發須提送水土保持計畫，並取得農委會水保局審查

### 4. 建照申請注意事項

- 主管機關：宜蘭縣政府工務處建築管理科
- 法規依據：宜蘭縣建築管理自治條例（民國107年8月13日）
- 建照申請需確認土地為都市計畫區或非都市土地（兩者法規不同）
- 溫泉區需另取得溫泉開發許可
- 農地建農舍嚴格限制非農業使用

### 5. 查詢系統

- [宜蘭縣都市計畫土地使用分區便民系統](https://ead.e-land.gov.tw/yilanusezoneforpublic/)
- [宜蘭縣建築管理自治條例](https://glrslaw.e-land.gov.tw/LawContent.aspx?id=FL023420)

## 十、連江縣（馬祖）

### 1. 都市計畫區使用分區建蔽率/容積率

連江縣各鄉（南竿、北竿、東引、莒光）均有獨立都市計畫，歷經多次通盤檢討放寬：

#### 南竿鄉都市計畫（通盤檢討後最新版）

| 使用分區 | 建蔽率（舊→新） | 容積率（舊→新） |
|---|---|---|
| 住宅區 | 60% → **80%** | 180% → **200%** |
| 商業區 | 80% → **80%** | 240% → **360%** |
| 旅館區 | 60% → **60%** | 120% → **360%** |
| 宗教專用區 | 50% → **60%** | 150% → **150%** |
| 公墓（納骨塔） | 20% → **30%** | 100% → **150%** |
| 殯儀館/火葬場 | — | 建蔽≤60%、容積≤120% |

#### 其他各鄉（北竿、東引、莒光）

各鄉設有獨立都市計畫，數值與南竿相近，依各鄉計畫書為準。通盤檢討後各鄉容積率均有提升趨勢。

### 2. 非都市土地（連江縣適用全國標準）

| 用地類別 | 建蔽率 | 容積率 |
|---|---|---|
| 甲種建築用地 | 60% | 240% |
| 乙種建築用地 | 60% | 240% |
| 丙種建築用地 | 40% | 120% |
| 丁種建築用地 | 70% | 300% |

### 3. 特殊規定（馬祖離島特性）

- **都市計畫適用範圍**：連江縣採「主要計畫＋細部計畫分離」模式，各鄉分別適用獨立都市計畫書，非全縣統一管制
- **保護區管制**：連江縣設有「連江縣都市計畫保護區土地使用審查要點」：
  - 申請基地面積上限3,000平方公尺（同一申請人合計不得超過1公頃）
  - 基地平均坡度不得超過30%（特殊情況30–55%需額外審查）
  - 申請基地需面臨已開闢道路或農路
  - 禁設地點：農地重劃區、保安林地、軍事禁建區、生態保育區、水源保護區、斷層帶
- **旅館區大幅放寬**：南竿旅館區容積率由120%調升至360%，反映馬祖觀光發展需求
- **預拌混凝土場**：保護區內允許設置（需審查核可），顯示離島建設實務需求
- **休閒農業**：風景區內可設置休閒農業及附屬設施
- **法規適用特殊性**：馬祖地區曾長期受戰地政務管轄，民國81年解除後才全面適用一般建築法規，部分舊建物有特殊豁免規定

### 4. 建照申請注意事項

- 主管機關：連江縣政府建設局（馬祖）
- 法規依據：連江縣建築管理相關自治條例、各鄉都市計畫書
- 離島建材需從台灣本島運送，建照申請需考量施工可行性
- 各鄉（南竿/北竿/東引/莒光）分別適用各自都市計畫，申請前須確認基地所屬計畫範圍
- 保護區開發需申請土地使用審查，程序較一般建照複雜
