## 法規架構概述

建築技術規則分為四編：

| 編別 | 名稱 | 主要內容 |
|------|------|---------|
| 總則編 | 建築技術規則總則編 | 名詞定義、適用範圍 |
| 設計施工編 | 建築設計施工編 | 建蔽率、容積率、高度、停車、無障礙設施等設計規範 |
| 構造編 | 建築構造編 | 結構安全、耐震、基礎設計 |
| 設備編 | 建築設備編 | 給排水、電氣、消防、昇降設備 |

> 本文件聚焦「設計施工編」中與土地開發、建照申請最密切相關的條文。

## 建蔽率相關條文

### 定義

**建蔽率**：建築面積（建築物地面層投影面積）佔基地面積之比率。

$$建蔽率 = \frac{建築面積}{基地面積} \times 100\%$$

### 第25條：建蔽率基本原則

| 項目 | 規定內容 |
|------|---------|
| 基本原則 | 基地之建蔽率，依都市計畫法及其他有關法令之規定 |
| 未規定地區 | 由直轄市、縣（市）政府訂定 |
| 法源 | 建築技術規則建築設計施工編第25條 |

**條文核心**：建蔽率不由建築技術規則直接訂定數值，而是依各地方都市計畫授權規定，建築技術規則作為技術性補充。

### 第26條：基地臨接公共設施之特例

| 條件 | 規定 |
|------|------|
| 基地臨接道路或永久性空地 | 達規定寬度比例以上之部分，得全部作為建築面積計算 |
| 目的 | 鼓勵基地整合，保留公共空間 |
| 法源 | 建築技術規則建築設計施工編第26條 |

### 第27條：高層建築空地加計規定

| 樓層數或高度 | 空地加計要求 |
|------------|------------|
| 超過5層或15公尺 | 每增加1層或4公尺，空地應增加2% |
| 計算方式 | 以基地面積為基準累計計算 |
| 上限 | 依各分區建蔽率反推可計算最高樓層數 |
| 法源 | 建築技術規則建築設計施工編第27條 |

**示例計算**：
- 基地面積1,000㎡，住宅區建蔽率60%
- 建築面積上限 = 600㎡（5層以下）
- 若建築8層（超過5層3層），空地應增加6%
- 調整後建蔽率上限 = 60% - 6% = 54%，建築面積上限 = 540㎡

### 第28條：騎樓面積不計入規定

| 項目 | 規定 |
|------|------|
| 適用分區 | 商業區 |
| 免計項目 | 法定騎樓面積 |
| 效果 | 不計入基地面積及建築面積 |
| 法源 | 建築技術規則建築設計施工編第28條 |

### 第29條：跨區基地計算原則

| 情況 | 處理方式 |
|------|---------|
| 基地跨越兩個以上使用分區 | 應按各分區規定分別計算空地、高度及建蔽率 |
| 分界線 | 以都市計畫分區線為準 |
| 法源 | 建築技術規則建築設計施工編第29條 |

## 容積率計算

### 定義

**容積率**：建築物各層樓地板面積之總和（容積總樓地板面積）與基地面積之比率。

$$容積率 = \frac{容積總樓地板面積}{基地面積} \times 100\%$$

**容積總樓地板面積**≠**總樓地板面積**（後者包含免計項目）

### 第161條：容積率定義與基本規定

| 項目 | 規定 |
|------|------|
| 容積率定義 | 建築物各層樓地板面積之總和除以基地面積之比率 |
| 適用前提 | 都市計畫法令或非都市土地使用管制規則已規定容積率之地區 |
| 計算基準 | 以「容積總樓地板面積」計算（非一般總樓地板面積） |
| 法源 | 建築技術規則建築設計施工編第161條 |

### 第162條：容積總樓地板面積計算方式

本條為容積計算核心條文，規定哪些面積「計入容積」、哪些「不計入容積」。

#### 計入容積之項目

| 計入項目 | 說明 |
|---------|------|
| 各樓層樓地板面積 | 含地下層（作為停車空間者除外） |
| 超出免計限額之陽台面積 | 超過該層10%部分須計入 |
| 超出免計限額之梯廳面積 | 超過部分須計入 |
| 夾層面積 | 原則計入容積 |
| 閣樓（超過限高者） | 計入容積 |

#### 不計入容積之項目（免計容積）

詳見下節「免計容積項目」完整清單。

## 免計容積項目

以下項目依建築技術規則第162條規定，不計入容積總樓地板面積：

### 主要免計項目一覽表

| 免計項目 | 免計條件與上限 | 備註 |
|---------|--------------|------|
| **陽台** | 每層不超過該層樓地板面積10% | 與梯廳合計不得超過15% |
| **梯廳（電梯廳）** | 每層不超過該層樓地板面積10% | 共用梯廳、昇降機廳 |
| **陽台+梯廳合計** | 合計不得超過該層樓地板面積15% | 超出部分計入容積 |
| **法定停車空間** | 全部免計 | 含機械停車設備空間 |
| **獎勵增設停車空間** | 全部免計 | 需符合獎勵規定 |
| **遮陽板** | 二分之一以上透空，深度在2公尺以下 | 需符合透空比例 |
| **露台** | 無頂蓋之露天平台 | 不作居室使用 |
| **法定騎樓** | 全部免計 | 需為法定設置 |
| **防空避難設備** | 依法設置者 | 兼作停車空間亦免計 |
| **屋頂突出物** | 依法設置，限樓梯間、水箱、機電設備 | 超出規定面積者計入 |
| **機電設備空間** | 總面積不超過基地容積10-15%（視條件） | 弱電室、發電機室等 |
| **安全梯間** | 依法設置之安全梯 | — |
| **雨遮** | 外牆附掛之短雨遮 | 深度及結構有限制 |
| **無障礙設施** | 依法設置之無障礙坡道、廁所 | — |
| **社區管理服務設施** | 集合住宅依規定設置者 | 有面積上限 |

### 陽台免計計算示例

```
建築物每層樓地板面積：500㎡
陽台面積：60㎡
梯廳面積：45㎡

合計 = 60 + 45 = 105㎡ = 21%（超過15%上限）
可免計面積 = 500㎡ × 15% = 75㎡
須計入容積面積 = 105 - 75 = 30㎡
```

### 重要說明

> **免計容積 ≠ 不占空間**：免計容積項目雖不計入容積率，但仍會增加建築物實際體積。建商常利用此機制增加實際可銷售面積，消費者應注意登記面積與實際使用面積之差異。

## 高度限制

### 第14條：臨路高度比計算（一般規定）

| 項目 | 規定 |
|------|------|
| 基本公式 | 建築物高度 ≤ 面前道路寬度 × 1.5 + 6公尺 |
| 斜線管制 | 依道路寬度、基地深度計算退縮斜線 |
| 法源 | 建築技術規則建築設計施工編第14條 |

**示例**：面前道路寬12公尺，建築物高度上限 = 12×1.5+6 = 24公尺

### 第23條：住宅區高度限制

| 條件 | 高度上限 | 層數上限 |
|------|---------|---------|
| 一般住宅區 | **21公尺** | **7層** |
| 臨接道路達15公尺寬，或面前空地寬達6公尺以上 | 可突破21公尺限制 | 視條件而定 |
| 鄰接永久性空地或道路合計達15公尺以上 | 可提高至一定限度 | — |
| 法源 | 建築技術規則建築設計施工編第23條 |

### 第24條：未實施容積管制地區高度限制

| 條件 | 高度上限 | 層數上限 |
|------|---------|---------|
| 一般規定 | **36公尺** | **12層** |
| 超越限制條件 | 需符合基地規模及道路寬度條件，送審核准 | — |
| 法源 | 建築技術規則建築設計施工編第24條 |

### 樓層高度限制

| 建築類型 | 地面一層高度上限 | 其他各層高度上限 |
|---------|----------------|----------------|
| 住宅、集合住宅 | **4.2公尺** | **3.6公尺** |
| 非住宅（商業、辦公等） | 依用途另定 | 依用途另定 |

### 特殊高度管制區

| 管制類型 | 說明 |
|---------|------|
| 航空噪音管制區 | 依民用航空法相關規定，高度另有限制 |
| 山坡地 | 依山坡地建築管制辦法另有規定 |
| 都市設計審議區 | 由地方政府訂定特別高度管制 |

## 停車空間規定

### 第59條：設置義務

| 適用情形 | 規定 |
|---------|------|
| 新建建築物 | 應依規定設置停車空間 |
| 改建建築物 | 變更用途或增建達一定規模者，應補設停車空間 |
| 設置標準 | 依建築物用途及樓地板面積，按附表規定設置 |
| 法源 | 建築技術規則建築設計施工編第59條 |

### 停車位設置標準

| 建築用途 | 設置標準 | 備註 |
|---------|---------|------|
| 第一類（商業、辦公、旅館等） | 每150㎡設1輛 | 以總樓地板面積計 |
| 第二類（住宅、集合住宅） | 每150㎡設1輛 | 以總樓地板面積計 |
| 第三類（學校、醫院等） | 依使用性質另定 | — |
| 機車停車位 | 汽車位數量1/4以上 | 部分縣市另有規定 |

### 第60條：停車位標準尺寸

| 項目 | 規格 |
|------|------|
| 標準車位（小型車） | 寬2.5公尺 × 長5.5公尺 |
| 殘障停車位 | 寬3.5公尺 × 長5.5公尺 |
| 雙車道寬度 | 5.5公尺以上 |
| 單車道寬度 | 3.5公尺以上（單向行駛） |
| 法源 | 建築技術規則建築設計施工編第60條 |

### 第61條：車道規定

| 項目 | 規格 |
|------|------|
| 車道坡度 | 不超過1/6（約9.5度） |
| 彎道內側曲線半徑 | 5公尺以上 |
| 出入口與道路交叉點距離 | 5公尺以上 |
| 法源 | 建築技術規則建築設計施工編第61條 |

### 第62條：停車空間環境規定

| 項目 | 規格 |
|------|------|
| 淨高 | 2.1公尺以上 |
| 通風 | 應設通風設備或自然採光通風 |
| 地面 | 應有適當舖裝 |
| 排水 | 應設排水設施 |
| 法源 | 建築技術規則建築設計施工編第62條 |

## 無障礙設施規定

### 第167條：設置義務與適用範圍

| 項目 | 規定 |
|------|------|
| 適用對象 | 供公眾使用之公共建築物 |
| 設置義務 | 應依第十章規定設置各項無障礙設施 |
| 設計規範 | 由中央主管建築機關另訂「建築物無障礙設施設計規範」 |
| 法源 | 建築技術規則建築設計施工編第167條 |

### 第170條：應設置無障礙設施之建築物類別

| 建築類別 | 舉例 |
|---------|------|
| 公共集會類 | 集會堂、電影院、表演廳 |
| 商業類 | 百貨公司、超市、旅館 |
| 休閒文教類 | 圖書館、博物館、運動中心 |
| 宗教殯葬類 | 寺廟、教堂、殯儀館 |
| 衛生福利更生類 | 醫院、診所、社福機構 |
| 辦公服務類 | 政府機關、銀行、郵局 |
| 住宿類 | 集合住宅（15戶以上） |

### 應設置之無障礙設施種類（共11種）

| 設施種類 | 說明 |
|---------|------|
| 1. 室外通路 | 從基地出入口至建築物入口之無障礙通路 |
| 2. 避難層坡道及扶手 | 替代階梯之坡道 |
| 3. 避難層出入口 | 寬度90公分以上，平坦或坡度1/50以下 |
| 4. 室內出入口 | 各樓層主要出入口 |
| 5. 室內通路走廊 | 寬度120公分以上 |
| 6. 樓梯 | 符合無障礙規格之樓梯 |
| 7. 昇降設備 | 無障礙電梯 |
| 8. 廁所盥洗室 | 無障礙廁所（依大便器數量設置） |
| 9. 浴室 | 供住宿使用者設無障礙浴室 |
| 10. 輪椅觀眾席 | 集會場所設置輪椅席 |
| 11. 停車空間 | 無障礙停車位 |

### 無障礙廁所設置數量規定

| 大便器數量 | 無障礙廁所應設置數量 |
|----------|-------------------|
| 1–19 個 | 1 處 |
| 20–29 個 | 2 處 |
| 30–39 個 | 3 處 |
| 每增加10個 | 增加1處 |
| 超過109個後 | 每增加10個增設1處 |

### 無障礙坡道規定

| 項目 | 規格 |
|------|------|
| 坡度上限 | 1/12（約4.8度） |
| 淨寬 | 90公分以上 |
| 平台 | 每隔9公尺設150×150公分以上之平台 |
| 扶手 | 兩側均應設扶手，高度75–85公分 |
| 止滑 | 坡面應設止滑設施 |

## 其他重要通則

### 採光與通風（第41–44條）

| 項目 | 規定 |
|------|------|
| 居室採光面積 | 居室樓地板面積之1/8以上（住宅） |
| 居室通風面積 | 居室樓地板面積之1/20以上 |
| 地下室 | 應設機械通風設備 |
| 法源 | 建築技術規則建築設計施工編第41–44條 |

### 樓梯規定（第33條）

| 項目 | 規格 |
|------|------|
| 最小寬度（住宅） | 75公分 |
| 最小寬度（公共建築） | 120公分以上 |
| 級高 | 不超過20公分 |
| 級深 | 不小於21公分 |
| 法源 | 建築技術規則建築設計施工編第33條 |

### 防火間隔（第110條）

| 項目 | 規定 |
|------|------|
| 同一基地不同建築物間距 | 依建築物高度及用途計算防火距離 |
| 防火牆 | 可取代防火間隔 |
| 法源 | 建築技術規則建築設計施工編第110條附近 |

## 名詞定義速查

| 名詞 | 定義 | 法源 |
|------|------|------|
| 建築面積 | 建築物地面層（含地下室頂板露出地面部分）外牆中心線以內之水平投影面積 | 第1條 |
| 總樓地板面積 | 建築物各層包括地下層、屋頂突出物及夾層等樓地板面積之總和 | 第1條 |
| 容積總樓地板面積 | 不包含免計容積項目之各層樓地板面積總和 | 第162條 |
| 建蔽率 | 建築面積佔基地面積之比率 | 第25條 |
| 容積率 | 容積總樓地板面積佔基地面積之比率 | 第161條 |
| 基地 | 依法供建築建造的土地 | 第1條 |
| 空地 | 基地面積扣除建築面積之餘地 | 第1條 |
| 騎樓 | 建築物地面層後退道路，作為行人通行之有頂蓋公共步行空間 | 第1條 |
| 陽台 | 直接由室內開向室外，且有頂蓋之平台 | 第1條 |
| 露台 | 直接由室內開向室外，無頂蓋之平台 | 第1條 |
| 夾層 | 樓層中設有中間樓地板之樓層（高度有限制） | 第1條 |
