## 1. EEWH 系統概述

### 1.1 基本資訊

| 項目 | 說明 |
|------|------|
| 制度名稱 | 台灣綠建築標章制度（EEWH Label） |
| 主管機關 | 內政部建築研究所 |
| 執行機構 | 財團法人台灣建築中心（TABC） |
| 建立年份 | 1999年（亞洲第一個熱帶/亞熱帶綠建築認證系統） |
| 分級制度建立 | 2007年（新增銅/銀/黃金/鑽石四級） |
| 氣候適應性 | 針對台灣亞熱帶高溫高濕氣候特性設計 |
| 指標數量 | 9大指標（1999年7項，2003年增至9項） |
| 認證等級數 | 5級（合格、銅、銀、黃金、鑽石） |

### 1.2 四大核心精神

```
E - Ecology（生態）：生物多樣性 + 綠化量 + 基地保水
E - Energy Saving（節能）：日常節能
W - Waste Reduction（減廢）：CO₂減量 + 廢棄物減量
H - Health（健康）：室內環境 + 水資源 + 污水垃圾改善
```

**來源：** [TABC 台灣建築中心 - 綠建築標章](https://www.tabc.org.tw/tw/modules/pages/gb01)、[維基百科 - 綠建築標章](https://zh.wikipedia.org/zh-tw/%E7%B6%A0%E5%BB%BA%E7%AF%89%E6%A8%99%E7%AB%A0)

## 2. 九大指標總覽

| 序號 | 指標名稱 | 類別 | 必要/選擇 | 核心評估重點 |
|------|---------|------|----------|------------|
| 1 | 生物多樣性指標 | 生態（E） | 選擇 | 綠地品質、生態棲地、物種豐富度 |
| 2 | 綠化量指標 | 生態（E） | 選擇 | 植栽固碳量、綠化面積 |
| 3 | 基地保水指標 | 生態（E） | 選擇 | 雨水滲透、地下水涵養 |
| 4 | 日常節能指標 | 節能（E） | **必要** | 建築外殼節能、設備能效 |
| 5 | CO₂減量指標 | 減廢（W） | 選擇 | 建材生產碳排量、再生材料使用 |
| 6 | 廢棄物減量指標 | 減廢（W） | 選擇 | 營建廢棄物減量、再利用率 |
| 7 | 室內環境指標 | 健康（H） | 選擇 | 光環境、音環境、空氣品質 |
| 8 | 水資源指標 | 健康（H） | **必要** | 節水器具、雨水回收、中水系統 |
| 9 | 污水垃圾改善指標 | 健康（H） | 選擇 | 污水處理、垃圾分類設施 |

> **關鍵規則：** 申請認證至少須通過4項指標，其中第4項（日常節能）與第8項（水資源）為**強制必要指標**，任何案件不得豁免。

**來源：** [888營建互聯網 - 綠建築EEWH評估系統](https://www.888civil.com/news/green-building-label)、[TABC](https://www.tabc.org.tw/tw/modules/pages/gb01)

## 3. 各指標詳細說明

### 3.1 生物多樣性指標（Biodiversity）

| 項目 | 說明 |
|------|------|
| 評估核心 | 生態綠地的品質與生物棲地豐富性 |
| 主要評估面向 | 基地綠地面積比例、喬木密度、生態池設置、鳥類/昆蟲棲地設計 |
| 實務做法 | 生態池、複層植栽、本土植物使用、人工濕地 |
| 適用說明 | 市中心高密度基地評分相對困難；郊區或大型校園最容易達標 |
| 類別 | 選擇指標 |

### 3.2 綠化量指標（Greening Quantity）

| 項目 | 說明 |
|------|------|
| 評估核心 | 建築基地植栽的固碳能力（以CO₂固定量計算） |
| 計算方式 | 喬木、灌木、草皮各有不同固碳係數，總固碳量除以基地面積得分 |
| 主要評估面向 | 屋頂花園、立體綠化、地面植栽、樹木保存 |
| 實務做法 | 屋頂綠化、垂直綠牆、陽台植栽、保留大型喬木 |
| 類別 | 選擇指標 |

### 3.3 基地保水指標（Site Water Retention）

| 項目 | 說明 |
|------|------|
| 評估核心 | 建築基地留存並滲透雨水的能力，維持地下水涵養 |
| 計算方式 | 透水鋪面面積比例、雨水花園、儲留設施蓄水量 |
| 主要評估面向 | 透水鋪面、雨水花園、滲透井、儲留設施 |
| 實務做法 | 透水磚、草皮、礫石鋪面、地下雨水貯留槽 |
| 豁免條件 | 山坡地建築、地下水位小於1公尺之地區、基地面積300m²以下 |
| 類別 | 選擇指標 |

### 3.4 日常節能指標（Daily Energy Saving）—必要指標

| 項目 | 說明 |
|------|------|
| 評估核心 | 建築物使用階段的能源消耗效率 |
| 分為兩大面向 | (A) 建築外殼節能設計；(B) 空調/照明/機電設備能效 |
| 外殼評估 | 外牆/屋頂隔熱性能、開窗比例、遮陽設計、自然通風 |
| 設備評估 | 冷氣能效(EER)、照明功率密度(LPD)、熱水器效率 |
| 強制門檻 | 外殼熱負荷指標（ENVLOAD）不得超過法定基準值 |
| 高雄氣候特點 | 高雄屬熱帶氣候，外殼隔熱與遮陽要求比北部更嚴格 |
| 類別 | **必要指標（無法豁免）** |

### 3.5 CO₂減量指標（CO₂ Reduction）

| 項目 | 說明 |
|------|------|
| 評估核心 | 建材生產過程產生的碳排放量（建造階段碳排） |
| 計算方式 | 各項建材用量 × 各材料碳排係數，總碳排量與基準值比較 |
| 主要評估面向 | 輕量化結構、鋼構使用率、建材循環再利用 |
| 實務做法 | 鋼結構（碳排係數較RC低）、木構造、再生骨材使用 |
| 類別 | 選擇指標 |

### 3.6 廢棄物減量指標（Waste Reduction）

| 項目 | 說明 |
|------|------|
| 評估核心 | 建築生命週期中廢棄物產生量的減少 |
| 主要評估面向 | 營建廢棄物再利用率、模組化施工、乾式工法使用 |
| 實務做法 | BIM精準用料、模組化板材、廢棄物分類再生 |
| 類別 | 選擇指標 |

### 3.7 室內環境指標（Indoor Environment）

| 項目 | 說明 |
|------|------|
| 評估核心 | 室內居住/工作環境的健康舒適性 |
| 三大面向 | (A) 音環境（隔音、混響）；(B) 光環境（採光、眩光）；(C) 空氣品質 |
| 空氣品質評估 | 甲醛含量、VOC揮發性有機物、通風換氣率 |
| 實務做法 | 低甲醛建材、自然通風設計、遮陽防眩光、隔音牆 |
| 類別 | 選擇指標 |

### 3.8 水資源指標（Water Resources）—必要指標

| 項目 | 說明 |
|------|------|
| 評估核心 | 建築物自來水節約使用與水資源循環再利用 |
| 主要評估面向 | 節水器具效率、雨水回收再利用、中水（廢水回收）系統 |
| 節水器具 | 省水馬桶（耗水量6L以下）、省水水龍頭、感應式沖洗閥 |
| 強制門檻 | 總樓地板面積達10,000m²以上新建建築物，強制設置雨水或生活雜排水回收系統 |
| 類別 | **必要指標（無法豁免）** |

### 3.9 污水垃圾改善指標（Sewage & Garbage Improvement）

| 項目 | 說明 |
|------|------|
| 評估核心 | 污水排放處理品質及固體廢棄物（垃圾）管理效率 |
| 主要評估面向 | 污水處理設施完整性、垃圾分類空間、回收設施設置 |
| 實務做法 | 截油槽、化糞池改善、垃圾分類儲放空間設計 |
| 類別 | 選擇指標 |

**來源：** [台灣建築中心TABC](https://www.tabc.org.tw/tw/modules/pages/gb01)、[888營建互聯網](https://www.888civil.com/news/green-building-label)、[易境永續設計顧問EEWH說明](https://www.greenjump.com.tw/en/service/eewh-service)

## 4. 分級認證門檻

### 4.1 五級認證制度

EEWH採用**得分蓋率（通過指標數/應評指標數之百分比）**進行分級：

| 認證等級 | 得分蓋率 | 說明 | 對應市場定位 |
|---------|---------|------|------------|
| 合格級 | < 30% | 基本通過（需達4項含2必要） | 法規最低門檻 |
| 銅級（Bronze） | 30% ~ 60% | 初級認證 | 一般商業建築常見 |
| 銀級（Silver） | 60% ~ 80% | 中級認證 | 優質住宅/辦公建築 |
| 黃金級（Gold） | 80% ~ 95% | 高級認證 | 企業旗艦/政府機關 |
| 鑽石級（Diamond） | ≥ 95% | 頂級認證 | 示範型綠建築 |

> **重要補充：** 得分蓋率的計算方式為各通過指標的得分加總，再與所有指標理論最高總分比較，而非單純計算通過指標數量。實際評分需透過各指標詳細計算。

### 4.2 最低申請門檻

- 至少通過 **4項指標**
- 其中**日常節能 + 水資源**為固定必要，不可豁免
- 另選任意 **2項** 以上

### 4.3 各等級對應指標數（概略參考）

| 等級 | 約需通過指標數 | 備註 |
|------|-------------|------|
| 合格級 | 4項 | 2必要 + 任2選擇 |
| 銅級 | 4-6項 | 需要高分通過多項指標 |
| 銀級 | 7-8項 | 多項指標達到優良標準 |
| 黃金級 | 8-9項 | 幾乎所有指標均達到優良 |
| 鑽石級 | 9項（高分） | 全部指標達到最高標準 |

> 注意：以上指標數為概略參考，實際取決於各指標得分計算。2項必要指標若得分極高，可能在較少指標數下就達到高等級。

**來源：** [888營建互聯網](https://www.888civil.com/news/green-building-label)、[維基百科 綠建築標章](https://zh.wikipedia.org/zh-tw/%E7%B6%A0%E5%BB%BA%E7%AF%89%E6%A8%99%E7%AB%A0)

## 5. 強制申請規定

### 5.1 建築技術規則強制綠建築基準（法規層面）

依建築技術規則建築設計施工編第17章第298條，以下為**法規強制要求**（非自願申請標章，而是設計必須符合基準）：

| 項目 | 適用條件 | 門檻 | 豁免情況 |
|------|---------|------|---------|
| 建築基地綠化 | 所有新建建築物 | 達到法定最低綠化量 | 基地面積300m²以下；個別農舍 |
| 建築基地保水 | 所有新建建築物 | 達到法定最低保水指標 | 山坡地；地下水位<1m；基地面積300m²以下 |
| 建築物節約能源 | 新建建築物超過1,000m²總樓地板面積 | 符合ENVLOAD基準值 | 機房；非營業倉庫；農舍500m²以下 |
| 雨水/生活雜排水回收 | 新建建築物總樓地板面積 ≥ **10,000m²** | 設置回收再利用系統 | 衛生醫療類建築 |

> 上述為技術規則強制要求，屬設計規範，與申請「標章」有別。

**來源：** [建築技術規則建築設計施工編第17章](https://twecoliving.blogspot.com/2021/03/17.html)

### 5.2 公有建築物強制申請標章

依「公有建築物綠建築標章推動使用作業要點」及相關政策：

| 類型 | 強制規定內容 | 生效時間 |
|------|------------|---------|
| 公有新建建築物 | 申請建造執照時，須同時申請候選綠建築證書 | 分年分階段實施 |
| 公有新建建築物（高能耗） | 優先納入，要求取得一定等級以上標章 | 自民國112年（2023年）7月1日起 |
| 政府補助改善案件 | 取得建築能效標示或候選建築能效證書 | 依補助辦法規定 |
| 其他指定建築物 | 由目的事業主管機關依權責訂定 | 依個別公告 |

### 5.3 建築能效標示（BERS）連動規定

> 自民國112年7月1日起，公有新建建築物申請**綠建築標章**時，須**同時申請建築能效標示（BERS）**。
> 兩者為互補制度，BERS聚焦能源效率等級（1~7+級），EEWH聚焦整體生態健康。

**來源：** [內政部建築研究所](https://www.abri.gov.tw/cp.aspx?n=804)、[綠建築標章申請審核認可及使用作業要點](https://glrs.moi.gov.tw/LawContent.aspx?id=FL050809)

## 6. 候選綠建築與正式標章差異

| 項目 | 候選綠建築證書 | 綠建築標章 |
|------|-------------|-----------|
| 申請時機 | 取得建造執照後、施工中 | 取得使用執照後（建築完工） |
| 意義 | 「準」綠建築認證，確認設計符合標準 | 正式認證，確認完工建築符合標準 |
| 有效期 | 施工期間（最長3年） | 5年（可申請展延） |
| 常見用途 | 申請建照前期確認、政府補助申請 | 不動產行銷、企業ESG報告 |
| 評定時間（廠房類） | 收件後30日內完成 | 收件後60日內完成 |
| 評定機構 | 台灣建築中心（TABC）委辦 | 內政部建築研究所核認 |

**來源：** [TABC官網](https://www.tabc.org.tw/tw/modules/pages/gb01)

## 7. 申請程序與流程

### 7.1 新建建築（候選綠建築 → 綠建築標章）

```
規劃設計階段
     ↓
確認目標等級（合格/銅/銀/黃金/鑽石）
     ↓
委託綠建築顧問進行指標計算
     ↓
取得建造執照（同步提送候選綠建築申請）
     ↓
施工中接受 TABC 評審（30日內）
     ↓
取得「候選綠建築證書」
     ↓
完工取得使用執照
     ↓
申請「綠建築標章」（60日內完成評審）
     ↓
取得正式「綠建築標章」（有效期5年）
```

### 7.2 既有建築改善（舊建築改善類）

```
現況調查與初步評估
     ↓
確認可改善之指標項目
     ↓
提出改善計畫（節能/節水/植栽等）
     ↓
執行改善工程
     ↓
申請「既有建築綠建築標章（EEWH-EB）」
```

### 7.3 申請單位

| 角色 | 說明 |
|------|------|
| 主管機關 | 內政部建築研究所 |
| 評定/執行機構 | 財團法人台灣建築中心（TABC） |
| 申請人 | 建築物起造人、所有權人或其代理人 |
| 輔助顧問 | 綠建築顧問公司（如澄毓、易境、匯續等） |

**申請單位聯絡：**
- 台灣建築中心（TABC）：https://www.tabc.org.tw
- 申請表單下載：https://www.tabc.org.tw/tw/modules/pages/easymenu04
- EEWH線上評估系統：https://eewh.tw

**來源：** [TABC官網](https://www.tabc.org.tw/tw/modules/pages/gb01)、[綠建築標章申請審核認可及使用作業要點](https://glrs.moi.gov.tw/LawContent.aspx?id=FL050809)

## 8. 評估類別與適用系統

EEWH依建築類型分為六大評估系統：

| 系統代號 | 適用類型 | 說明 |
|---------|---------|------|
| EEWH-BC | 基本型與住宿類（Basic & Clinic） | 一般住宅、辦公、醫療機構 |
| EEWH-RS | 住宿類（Residential） | 集合住宅（公寓大廈）為主 |
| EEWH-GF | 廠房類（General Factory） | 工廠、廠房、倉庫 |
| EEWH-EB | 舊建築改善類（Existing Building） | 已取得使用執照之既有建築 |
| EEWH-SC | 社區類（Settlement & Community） | 整體社區開發計畫 |
| EEWH-EC | 教育設施類（Educational Complex） | 學校、大學校園 |

## 9. 建築技術規則強制綠建築基準

台灣於民國94年（2005年）將綠建築政策**法制化**，成為全球首創將綠建築基準納入法令強制執行的國家。

### 9.1 第17章適用建築類型（強制符合，非申請標章）

### 9.2 綠建材使用規定

依建築技術規則第17章相關規定：
- 新建建築物內裝材料須使用一定比例之**綠建材**（低甲醛、低VOC）
- 適用於：室內裝修工程（地板、牆面、天花板）
- 比例門檻：內裝面積使用綠建材比例需達一定標準（依用途別有所不同）

## 12. 常見問答（FAQ）

**Q1：申請 EEWH 標章需要花多少費用？**
> A：評定費用依建築類型與規模而定，由台灣建築中心（TABC）公告收費標準。一般住宅小型案件約數萬元，大型商業或廠房案件可能達十萬元以上。顧問費用另計。

**Q2：標章效期到期後如何延續？**
> A：標章有效期為5年，到期前6個月可申請展延，需重新評估確認建築仍符合標準。

**Q3：高雄的建築申請 EEWH 有什麼特殊考量？**
> A：高雄屬熱帶氣候（Köppen Aw型），全年高溫、夏季多雨。日常節能指標中的外殼熱負荷（ENVLOAD）基準值比台北嚴格；基地保水指標因颱風季強降雨，設計上需特別注意。

**Q4：EEWH 與建築能效標示（BERS）有何不同？**
> A：EEWH評估建築整體生態健康表現（9大指標全面評估）；BERS（Building Energy Rating System）專注評估建築能源效率，以1~7+等級標示。自112年起公有建築申請EEWH須同時申請BERS。

**Q5：舊建築如何申請？**
> A：使用EEWH-EB（既有建築改善類）系統申請，著重在可改善之節能、節水、植栽等措施。不需符合全新建築的所有指標要求。

**Q6：取得鑽石級有多困難？**
> A：台灣截至近年的鑽石級認證建築相對稀少，主要為政府示範性建築（如衛福部衛生福利大樓）或企業旗艦廠房。鑽石級要求所有9大指標均達到優良標準（得分蓋率95%以上），設計複雜度高且建造成本顯著增加。

**Q7：EEWH 對容積率有獎勵嗎？**
> A：部分都市計畫特別規定區域（如某些科學園區、都更案）可提供容積獎勵，但非全面性規定。建築技術規則第301條亦規定中央主管機關「得以增加容積」鼓勵綠建築設計，但截至本文件更新日，該條款施行日期尚未確定。
