# 全國都市更新與危老重建法規整理
> 法源：都市更新條例（民國113年10月25日修正）、都市危險及老舊建築物加速重建條例（民國114年修正草案）
> 主管機關：內政部國土管理署（都市更新組）
> 資料查核日期：2026-03-26（三次獨立查核：全國法規資料庫 × HouseFeel × 安信建經 × 耕薪建設）
> ✅ 三次查核通過

---

## 一、都更 vs 危老重建：核心差異對照表

| 比較項目 | 都市更新（都更） | 危老重建 |
|----------|----------------|----------|
| **適用法源** | 都市更新條例 | 都市危險及老舊建築物加速重建條例 |
| **主管機關** | 各縣市都市更新主管機關 | 各縣市建築主管機關 |
| **申請資格** | 都市計畫範圍內，指定更新地區 | 都市計畫範圍內，危險或老舊建物 |
| **基地面積** | 通常需達一定規模（各縣市不同） | **無最小面積限制**（單戶即可） |
| **所有權人同意比例** | 重建：需**75%以上**（面積與人數雙門檻）；整建維護：需**2/3以上** | **100%同意**方可執行 |
| **容積獎勵上限** | 基準容積**1.5倍**，或原容積 + 基準容積0.3倍 | 基準容積**1.3倍**，或原容積1.15倍 |
| **審查期限** | 較長（涉及公聽會、審查委員會） | **30日內**完成重建計畫審查 |
| **程序複雜度** | 高（需公開展覽、公聽會、同意書） | 低（文件齊備即可） |
| **政府介入程度** | 高（可強制徵購）| 低（私人自主） |
| **稅賦優惠** | 更新期間地價稅、房屋稅減免 | 更優惠（見第五節） |
| **申請期限** | 無期限限制 | 116年5月31日前申請（時程獎勵已於114年歸零）|
| **適用情境** | 大型街廓整體更新、公共設施整備 | 個別危險或老舊建物快速重建 |

## 二、都市更新條例重點（113年修正版）

### 2.1 更新地區劃定條件

| 條件類型 | 具體條件 |
|----------|----------|
| **建物條件** | 建築物窳陋且非防火構造或鄰棟間隔不足 |
| **設施條件** | 建築物因年代久遠有傾頹或朽壞之虞 |
| **公益條件** | 建築物未符合都市應有之機能 |
| **都市計畫** | 政府指定為更新地區者 |

### 2.2 容積獎勵項目與上限（都市更新建築容積獎勵辦法）

| 獎勵類型 | 獎勵幅度 | 上限說明 |
|----------|----------|----------|
| 都市計畫指定更新 | 依個案計算 | 基準容積**1.5倍** |
| 歷史建物保存 | 依保存面積比例 | 含入獎勵總量 |
| 綠建築標章 | 2%～10% | 含入獎勵總量 |
| 智慧建築標章 | 2%～10% | 含入獎勵總量 |
| 無障礙設施 | 1%～5% | 含入獎勵總量 |
| 社會住宅捐贈 | **免計容積** + 獎勵 | 113年新增，不限額度（總量管制下） |
| **總上限（一般）** | 不得超過基準容積**1.5倍** | 或原建築容積 + 基準容積×0.3 |

### 2.3 113年修法重點

| 修法項目 | 修法前 | 修法後（113年10月25日） |
|----------|--------|----------------------|
| 原容積重建適用對象 | 容積管制前**已興建完成**的合法建物 | 容積管制前**已申請建照**的合法建物（擴大） |
| 社會住宅獎勵 | 未有獨立機制 | 新增捐贈社宅免計容積 + 另給容積獎勵 |
| 更新同意比例 | 重建75%、整建2/3 | 維持原比例（依類型不同） |

### 2.4 都更申請流程

```
1. 確認更新地區（政府劃定或自行申請劃定）
   ↓
2. 擬定都市更新事業計畫（可委託實施者）
   ↓
3. 公開展覽（30日）
   ↓
4. 公聽會（至少2次）
   ↓
5. 召開審議會審查
   ↓
6. 取得土地及建物所有權人同意書
   （重建：面積＋人數均達75%；整建：2/3）
   ↓
7. 報請主管機關核定
   ↓
8. 拆除、施工、重建
   ↓
9. 申請使用執照
   ↓
10. 分配權益（依權利變換計畫）
```

## 三、危老重建條例重點

### 3.1 申請資格條件（需符合其中之一）

| 條件類型 | 具體說明 |
|----------|----------|
| **危險建築** | 經主管機關通知限期拆除；或結構安全性能評估未達最低等級（D級） |
| **老舊建築（無電梯）** | 屋齡**30年以上**，耐震評估未達一定標準，且改善費用高於重建費用 |
| **老舊建築（有電梯）** | 屋齡**30年以上**，電梯已達年限，且設備全面更新改善無實益 |
| **位置條件** | 須位於**都市計畫範圍內** |
| **建築合法性** | 需為**合法建築物**（有使用執照或歷史地籍圖可查） |
| **非保存對象** | 非古蹟、歷史建築，或無保存價值認定 |

### 3.2 容積獎勵詳細項目

| 獎勵類別 | 細項 | 獎勵幅度 |
|----------|------|----------|
| **危老資格獎勵** | 建物評估等級 D 級 | +10% |
| **危老資格獎勵** | 建物評估等級 E 級 | +8% |
| **危老資格獎勵** | 建物評估等級 F 級 | +6% |
| **耐震設計標章** | 耐震設計達一定標準 | +2%～+10% |
| **綠建築標章** | 合格級以上 | +2%～+6% |
| **智慧建築標章** | 合格級以上 | +2%～+6% |
| **無障礙住宅標章** | 達標 | +3%～+5% |
| **結構安全評估優化** | 超出最低標準 | +2%～+4% |
| **時程獎勵** | 已於114年5月9日歸零 | 0%（原最高8%） |
| **規模獎勵** | 基地面積達一定規模 | 視規模而定（最高10%） |
| **加值獎勵總上限** | 以上各項合計 | **不得超過30%** |

### 3.3 容積獎勵上限規定

| 規模 | 容積獎勵上限 | 說明 |
|------|-------------|------|
| 一般案件 | 基準容積**×1.3倍** | 或原建築容積×1.15倍（取有利者） |
| 未達500m²小基地（114年修正草案） | 基準容積**×1.2倍** | 或原建築容積×1.1倍 |
| 超過500m²基地 | 基準容積**×1.3倍** | 維持現行標準 |

### 3.4 稅賦優惠

| 優惠項目 | 適用期間 | 內容 |
|----------|----------|------|
| 地價稅 | 重建期間 | **全免** |
| 房屋稅 | 重建期間 | **全免** |
| 地價稅 | 完工後2年內 | **減半徵收** |
| 房屋稅 | 完工後2年內 | **減半徵收** |
| 地價稅 | 第3年起 | 恢復正常 |
| 房屋稅 | 未移轉者，完工後 | 可再**減半徵收10年** |
| 契稅 | 重建後第一次移轉 | 減徵（依各縣市規定） |

### 3.5 危老重建申請流程

```
1. 確認建物資格（危險或30年以上老舊）
   ↓
2. 取得過半戶數同意（啟動評估程序）
   ↓
3. 委託辦理結構安全初評
   ↓
4. 取得100%土地及建物所有權人同意書
   （這是危老與都更最大差異）
   ↓
5. 備妥重建計畫書、圖說、說明書
   ↓
6. 向主管機關提出重建計畫審查申請
   ├── 台北市：建築管理工程處
   ├── 新北市：城鄉發展局（都市更新處）
   ├── 台中市：建築管理處
   ├── 高雄市：建築管理處
   └── 其他縣市：工務局/建設處
   ↓
7. 審查（原則30日內完成）
   ↓
8. 核准重建計畫
   ↓
9. 申請拆除執照（含繳交容積獎勵保證金）
   ↓
10. 申請建造執照（核准後360日內）
    ↓
11. 拆除施工
    ↓
12. 申請使用執照
    ↓
13. 取得相關標章，申請退還保證金
```

### 3.6 容積獎勵保證金計算

| 獎勵幅度 | 保證金計算方式 |
|----------|---------------|
| 超過基準容積6%者 | 公告現值 × 0.45 × 申請獎勵樓地板面積 |
| 未超過基準容積6%者 | 上述金額 × 0.5（減半） |
| **退款條件** | 領使用執照後2年內取得相關標章並提交證明 → 無息退還 |
